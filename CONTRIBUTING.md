# Contributing to AI Auto-Reply Agent

We love your input! We want to make contributing to this project as easy and transparent as possible, whether it's:

- Reporting a bug
- Discussing the current state of the code
- Submitting a fix
- Proposing new features
- Becoming a maintainer

## Development Process

1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the UI remains responsive and follows the premium design language.
5. Issue that pull request!

## Code of Conduct

Please be respectful and professional in all interactions.

## Design Principles

- **Speed**: Responses should feel instantaneous.
- **Clarity**: Intents should be clearly tagged.
- **Safety**: Always provide an escalation path for the user.
